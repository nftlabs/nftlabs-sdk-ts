### Testing

#### Running all tests

```bash
$ yarn run test:all
```

#### Running a specific test file

```bash
$ yarn run test test/pack.test.ts
```

#### Running a matching set of tests

```bash
$ yarn run test test/pack.test.ts --grep "should allow you to buy a pack"
```
